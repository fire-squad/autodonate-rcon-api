Simple RCON API for Autodonate

## Особенности

- Бесплатно! Мы не попросим ни копейки за использование!
- Добавьте свое!


## Установка

```bash
pip install autodonate-rcon-api
```

## Пример

```py
from autodonate_rcon_api.example import some_function

print(some_function(3, 4))
# => 7
```
